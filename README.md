## TECHSTACK

- React Hooks
- React Router DOM
- Material UI
- JSON DB localhost

## How to install

- To use this app you need to install Node.js v14
- Fork and clone this repository
- Run 'npm install' in your terminal
- Run 'npm run json:server' for db json server
- Run 'npm start'
